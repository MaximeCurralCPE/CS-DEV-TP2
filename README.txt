Bonjour !

Vous trouverez la version console du pendu dans pendu_console.py, qui utilise les fonctions de pendu_console_lib.py
La deuxième version console est dans pendu_console_v2.py, et ses fonctions dans pendu_lib_v2.py

La version graphique se trouve dans pendu_graphique.py
Les images de pendu sont dans le dossier images

Les mots sur lesquels vous pouvez tomber en jeu sont dans mots.txt, et le meilleur score est dans score.txt

Vous pouvez retrouver tous ces éléments ici : https://github.com/MaximeCurralCPE/TP2.git

Bonne lecture !
